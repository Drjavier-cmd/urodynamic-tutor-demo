# Matriz de competencias

## Competencias transversales

| ID | Competencia | Evidencia esperada |
|---|---|---|
| question | Pregunta | Formula qué intenta responder o descartar el estudio. |
| phase | Fase | Ubica almacenamiento, maniobra, transición o vaciado. |
| quality | Calidad | Decide si el segmento es interpretable y por qué. |
| signal | Señal | Distingue Pves, Pabd, Pdet, flujo, EMG y eventos. |
| event | Evento | Relaciona cambios con tos, Valsalva, posición, fuga o sensación. |
| inference | Inferencia | Propone una explicación funcional proporcional. |
| limit | Límite | Declara qué no puede concluirse. |
| response | Respuesta | Redacta una respuesta clínica breve y prudente. |

## Dominios obligatorios de evaluación

Cada pretest y postest debe incluir al menos dos reactivos de:

1. calidad técnica;
2. almacenamiento;
3. vaciado;
4. coordinación;
5. representatividad;
6. límites de interpretación.

## Regla de dominio

- `Sin evidencia`: no existen intentos válidos.
- `Inicial`: menos de 60% de respuestas correctas.
- `En desarrollo`: entre 60% y 79%.
- `Dominio`: 80% o más, con al menos dos evidencias.

Los textos libres del caso final no se califican automáticamente como correctos.
Se comparan con una rúbrica y requieren revisión humana para certificación.
